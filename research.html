A Behavioral Analysis of Ransomware in Active Directory: A Case Study of BlackMatter, Conti, LockBit, and Midnight

Ransomware remains a pervasive threat in enterprise Active Directory (AD) environments. We analyze Windows Event Logs from attacks by four families—BlackMatter, Conti, LockBit, and Midnight—running 20 samples in a controlled Proxmox‑based AD lab. Using n‑gram sequence modeling and behavioral feature extraction, we uncover family‑specific event sequences (e.g., service control and credential validation) that indicate ransomware activity, laying groundwork for behavior‑based AD detection.

Ransomware Evolution: Unveiling Patterns Using HDBSCAN

We leverage Windows API calls and PE header features to develop family‑level signatures via hierarchical density‑based clustering (HDBSCAN) with Jaccard similarity across 1.1M malicious samples. The approach isolates distinct API‑usage patterns for families like GandCrab, WannaCry, Cerber, Gotango, and CryptXXX. The resulting signatures achieve 98.34% precision and 99.72% specificity on an independent set, demonstrating scalable, robust detection of known ransomware families.

Searching for Selfie in TLS 1.3 with the Cryptographic Protocol Shapes Analyzer

Analysis of TLS 1.3 handshake modes (1‑RTT authenticated, PSK resumption with optional (EC)DHE, and 0‑RTT) in a reductionist security model. We tag derived session keys by properties (authentication, forward secrecy, replay resistance, usage), and show that TLS 1.3 modes establish session keys with desired security properties under standard assumptions.

Trends in Malware

We study malware family evolution using SoReL‑20M. Samples are embedded (EMBER), normalized, and reduced (PCA), with pairwise distances forming a minimum spanning tree to surface familial similarity sequences. Early results highlight large Trojan clusters (and Ransom/Downloader subsets), motivating deeper family‑level analysis.

Malware Antivirus Scan Pattern Mining via Tensor Decomposition

AV vendor labels are inconsistent across providers. Using CP‑APR tensor decomposition on multi‑dimensional AV scan data reveals vendor‑specific reporting patterns, improving interpretability versus traditional ML. This aids downstream labeling consistency for malware classification and clustering.
